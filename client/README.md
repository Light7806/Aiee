# Dashboard Layer

The dashboard is the operator view for Aiee.

It will show what the bot-defense system is seeing in real time.

## Planned Panels

- Traffic volume.
- Blocked/challenged/allowed request counts.
- Bot score distribution.
- Top offending IPs.
- Signal breakdown.
- Whitelist status.

The current React files are placeholders. They define the dashboard layout before the live analytics API is implemented.
